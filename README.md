# CountryPeek

A React application to search, explore, and save countries using the RestCountries API.

## 🌍 Live Demo
https://akshargk.github.io/country-peek/

## ✨ Features
- 🔍 Search countries by name
- 🌎 Filter by region
- ↕️ Sort by name and population
- 📄 Country detail page
- ❤️ Add/remove favourites (saved in localStorage)
- 🌙 Light/Dark mode toggle
- 📱 Responsive design (mobile friendly)

## 🛠 Tech Stack
- React (Vite)
- React Router
- Context API
- useReducer
- REST API (RestCountries)
